# Internal documentation (maintainer-only)

This folder contains **internal and historical** material: reviews, plans, analyses, prompt templates, archive, and completed work. It is for the maintainer only and is not part of the public contributor-facing documentation.

- **Public docs:** See `docs/` and [docs/DOCUMENTATION_INDEX.md](../docs/DOCUMENTATION_INDEX.md).

## Subdirectories

| Folder | Contents |
|--------|----------|
| `prompts/` | AI agent prompt templates, review prompts, task prompts, workflow instructions (e.g. AGENT_STRICT_CONSTRAINTS, TaskmasterPrompt, clang-tidy/Sonar workflows) |
| `analysis/` | Dated code reviews, performance analyses, improvement backlogs, tooling investigations |
| `design/` | Internal design reviews, in-progress analyses, library upgrade notes |
| `plans/` | Maintainer checklists, publish workflows, roadmap items |
| `Historical/` | Completed code reviews and resolved issue explanations |
| `archive/` | Superseded analyses and implementation plans |
| `review/` | Dated review bundles |
| `research/`, `optimization/`, `bugs/`, `fixes/`, `futures/`, `features/`, `testing/`, `ui-ux/`, `boost/`, `deduplication/`, `examples/` | Topic-organized internal notes |
